Class links.Graph.StepNumber

StepNumber The class StepNumber is an iterator for numbers. You provide a start and end value, and a best step size. StepNumber itself rounds to fixed values and a finds the step that best fits the provided step. If prettyStep is true, the step size is chosen as close as possible to the provided step, but being a round value like 1, 2, 5, 10, 20, 50, .... Example usage: var step = new links.Graph.StepNumber(0, 10, 2.5, true); step.start(); while (!step.end()) { alert(step.getCurrent()); step.next(); } Version: 1.0
Defined in: graph.js.
